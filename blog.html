Important FAQ:

1. What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of HTML documents as a tree, allowing scripts to manipulate content dynamically.

2. How do you select an element from the DOM?

To select an element from the DOM, use methods like document.getElementById(), document.querySelector(), or document.getElementsByClassName() to access elements by ID, class, tag, or CSS selectors.

3. What is event delegation in the context of the DOM, and why is it useful?

Event delegation in the DOM allows a parent element to handle events from its child elements. It improves performance by reducing event listeners and simplifies dynamic content handling.

4. How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes, use setAttribute() or direct properties like element.id. For styles, access element.style or use classList to add, remove, or toggle CSS classes.
